# Change Log

All notable changes to the "console-highlighter" extension will be documented in this file.

## [0.1.0] - 2026-04-16

### Changed

- Updated console.log color to neon green `rgba(0, 255, 106, 0.8)` for better visibility
- Simplified status bar text to "CH"

## [0.0.2] - 2026-04-16

### Added

- Support for `console.dir` with slate color highlighting
- Neon green default color for `console.log`

## [0.0.1] - 2026-04-15

### Added

- Initial release
- Highlight console.log, console.info, console.warn, console.error, console.debug
- Support for JavaScript (.js, .mjs, .cjs)
- Support for TypeScript (.ts)
- Support for React (.jsx, .tsx)
- Configurable colors for each console method
- Real-time highlighting as you type
